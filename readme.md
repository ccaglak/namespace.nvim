## Namespace.nvim

Neovim Php Namespace Resolver

## Install

```lua

{'ccaglak/namespace.nvim'}

```

## Requires

-   pleanery.nvim
-   rg

## Basic Usage

-   `:GetAllClasses` Finds all classes

-   `:GetClass` gets class under cursor

-   `:SortClass` sorts classes
